## Requirements
- Node.js (LTS recommended)
- npm (comes with Node.js)

## Dependencies
- devDependencies:
  - tailwindcss `^3.4.19`

## Setup
1. Install dependencies:
	```bash
	npm install
	```
2. Build the CSS once:
	```bash
	npm run build
	```
3. Open `index.html` in a browser to view the site.

## Development (optional)
Watch for changes and rebuild automatically:
```bash
npm run watch
```

## Project Structure
- `src/input.css` - Tailwind source file
- `dist/output.css` - Generated CSS (do not edit manually)
- `*.html` - Static pages
- `resources/` - Images and assets
